---
title: Person Dataset
---

The `person` dataset provides commands for generating a consistent set of random personal data (name, contact info, and demographics for a single fake person). Below is a detailed explanation of all available properties and their parameters.

## Available Properties

### avatar
Generates a random avatar image URL for the person.

**Usage:**

```bash
bogus dataset person.avatar
```

### dateOfBirth
Generates a random date of birth, formatted as `yyyy-MM-dd`.

**Usage:**

```bash
bogus dataset person.dateOfBirth
```

### email
Generates a random email address for the person.

**Usage:**

```bash
bogus dataset person.email
```

### firstName
Generates a random first name.

**Usage:**

```bash
bogus dataset person.firstName
```

### fullName
Generates a random full name.

**Usage:**

```bash
bogus dataset person.fullName
```

### gender
Generates a random gender (e.g., Male, Female).

**Usage:**

```bash
bogus dataset person.gender
```

### lastName
Generates a random last name.

**Usage:**

```bash
bogus dataset person.lastName
```

### phone
Generates a random phone number for the person.

**Usage:**

```bash
bogus dataset person.phone
```

### userName
Generates a random username for the person.

**Usage:**

```bash
bogus dataset person.userName
```

### website
Generates a random personal website domain for the person.

**Usage:**

```bash
bogus dataset person.website
```
