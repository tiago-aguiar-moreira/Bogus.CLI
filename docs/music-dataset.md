title: Music Dataset

The `music` dataset provides commands for generating random music-related data. Below is a detailed explanation of all available properties and their parameters.

## Available Properties

### genre
Generates a random music genre (e.g., Rock, Jazz, Classical).

**Usage:**

```bash
bogus dataset music.genre
```
