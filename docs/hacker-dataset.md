The `hacker` dataset provides a variety of commands for generating random tech/hacker jargon. Below is a detailed explanation of all available properties and their parameters.

## Available Properties

### abbreviation
Generates a random tech abbreviation (e.g., HTTP, SQL, TCP).

**Usage:**

```bash
bogus dataset hacker.abbreviation
```

### adjective
Generates a random hacker adjective (e.g., cross-platform, open-source).

**Usage:**

```bash
bogus dataset hacker.adjective
```

### ingVerb
Generates a random hacker verb in the gerund form (e.g., parsing, indexing).

**Usage:**

```bash
bogus dataset hacker.ingVerb
```

### noun
Generates a random hacker noun (e.g., protocol, interface, driver).

**Usage:**

```bash
bogus dataset hacker.noun
```

### phrase
Generates a random hacker phrase combining multiple jargon elements.

**Usage:**

```bash
bogus dataset hacker.phrase
```

### verb
Generates a random hacker verb (e.g., bypass, compress, generate).

**Usage:**

```bash
bogus dataset hacker.verb
```
